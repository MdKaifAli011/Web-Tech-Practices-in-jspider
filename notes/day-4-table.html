Table in HTML

A table is a collection of rows and columns, similar to a spreadsheet. To create a table in HTML, you use the <table> and </table> tags. You create the rows and columns using specific HTML tags.

Basic Tags for Creating Tables

  • <table></table> - Creates the table container.
  • <caption></caption> - Defines a title or caption for the table.
  • <tr></tr> - Defines a table row.
  • <th></th> - Defines a header cell (for table headings).
  • <td></td> - Defines a table data cell (for regular data).

Example of a Table:

Example Table
Product Price Quantity
Apple $2 5
Banana $1 8
Total Price $15

Cell Spanning in HTML Tables:

In HTML, you can span cells in both column and row directions.

1. colspan:

colspan is used to span cells across multiple columns.

<td colspan="2">Total Price</td>

2. rowspan:

rowspan is used to span cells across multiple rows.

<td rowspan="2">Cell Spanning</td>

Example with Colspan and Rowspan:

Table with Spanning
Product Price Quantity
Apple $2 5
Banana $1 $8
Total $10

Other Table Elements:

  • <thead> - Defines the header section of the table.
  • <tbody> - Defines the body section of the table.
  • <tfoot> - Defines the footer section of the table.

Remove Double Border in Tables:

To remove the double border and change it to a single line, use the rules="all" attribute in the <table> tag.

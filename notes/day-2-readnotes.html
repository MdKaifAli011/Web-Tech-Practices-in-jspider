Understanding HTML Attributes

What Are HTML Attributes?

Attributes in HTML represent the characteristics and properties of a tag. They modify the behavior of an HTML element or provide additional information about the element.

Attributes are written inside the opening tag of an element. They typically come in a key-value pair format.

Why Use Attributes?

We use attributes to define specific behavior or to style a tag in a described manner. Without attributes, tags would be very generic and not perform as expected for certain tasks.

Where Should Attributes Be Written?

Attributes should always be written inside the opening tag. If you apply an attribute to a closing tag, it won't work as expected.

Examples of HTML Attributes

Absolute Path

The absolute path is the full path to a file, starting from the root directory of the file system. It is used to reference files from anywhere in the directory structure.

<a href="/images/photo.jpg">Link to Image</a>

Relative Path

The relative path points to a file from the current directory. It's used when the file you are referencing is located in the same directory or subdirectory.

<a href="photo.jpg">Link to Image</a>

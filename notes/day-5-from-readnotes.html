HTML Form Example

1. Introduction

Forms in HTML are used to collect input from the user. The <form> tag allows you to collect data from a user by including various input fields like text, email, password, etc.

2. Basic HTML Form Structure

To create a form in HTML, you must use the <form> tag. Within the form, you use <input> tags to create individual input fields. Here's an example:

<form action="/submit" method="post">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username" required>

    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>

    <label for="password">Password:</label>
    <input type="password" id="password" name="password" required>

    <input type="submit" value="Submit">
</form>

3. Explanation

  • <form>: Used to define the form structure. It has attributes like action and method.
  • <input>: Used to define input fields. The type attribute defines the input type (e.g., text, email, password).
  • <label>: Provides a label for each input field, improving accessibility and user experience.
  • <input type="submit">: Creates a button that submits the form.

4. Key Points

  • Forms are used to collect user input.
  • The <form> tag encloses all the input elements.
  • <input> tags are used to create fields for users to interact with.
  • To submit the form data, the <input type="submit"> tag is used.

5. Example Form

This is how a simple HTML form might look when rendered:

<form action="/submit" method="post">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username">

    <label for="email">Email:</label>
    <input type="email" id="email" name="email">

    <label for="password">Password:</label>
    <input type="password" id="password" name="password">

    <input type="submit" value="Submit">
</form>

Note: Make sure to replace /submit with the correct action URL where you want to send the form data. The method attribute specifies how the data will be sent, usually POST or GET.

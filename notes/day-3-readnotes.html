Understanding Lists in HTML

What is a List in HTML?

A list in HTML is a collection or group of related items. Lists help organize content and are often used to display items such as menu options, steps in a process, or key points in a document.

Types of Lists

There are three primary types of lists in HTML:

1. Ordered List (<ol>)

An ordered list is used to display items in a specific order. The items in an ordered list are numbered automatically.

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

2. Unordered List (<ul>)

An unordered list displays a list of items with bullet points, rather than numbers.

<ul>
    <li>Item A</li>
    <li>Item B</li>
    <li>Item C</li>
</ul>

3. Description / Dictionary List (<dl>)

A description list (or dictionary list) is used to define terms and their descriptions. This is typically used for glossaries or FAQs.

Example of Description List

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language, used for creating webpages</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets, used for styling webpages</dd>
</dl>

Attributes of the Input Tag

1. Type Attribute

The type attribute is used to specify the kind of data the input field accepts.

Syntax: <input type="___________________________">

Possible Values for the Type Attribute:

  • text - Accepts plain text input
  • tel - Accepts a telephone number
  • email - Accepts an email address
  • password - Accepts a password (hidden text)
  • file - Allows file upload
  • date - Accepts a date input
  • time - Accepts time input
  • datetime-local - Accepts date and time input
  • color - Accepts color input
  • range - Accepts a range (e.g., slider)
  • search - Accepts search text
  • url - Accepts a URL input
  • submit - Defines a submit button
  • reset - Defines a reset button
  • number - Accepts numerical input
  • radio - Defines a radio button
  • checkbox - Defines a checkbox

Example Usage:

For a simple text input field:

<input type="text">

For a password input field:

<input type="password">

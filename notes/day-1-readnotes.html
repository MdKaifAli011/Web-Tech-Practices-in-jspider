Understanding HTML Tags

Opening and Closing Tags

A tag in HTML is a keyword enclosed by angular brackets (<>) that defines elements of a webpage.

When you enclose a keyword with angular brackets <div>, it's known as the opening tag.

When you use the same keyword but with a forward slash (/) before it, enclosed in angular brackets </div>, it's called the closing or ending tag.

Types of Tags

In HTML, there are two main types of tags:

1. Pair Tags

Pair tags are those that require both an opening tag and a closing tag. These tags are used to define content that has a start and end. For example:

<p>This is a paragraph.</p>

The <p> tag starts the paragraph, and the </p> tag ends it.

2. Unpaired Tags

Unpaired tags are self-closing tags that do not require a closing counterpart. These tags define a single element or function. For example:

<br />

The <br> tag, which is used for a line break, doesn't need a closing tag.

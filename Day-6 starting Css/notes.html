CSS Selectors Guide

1) Simple Selectors

  • #id → Selects an element with a specific ID.
  • .class → Selects elements with a specific class.
  • element → Selects all elements of a specific type (e.g., p, div).
  • * (Universal) → Selects all elements.
  • , (Group) → Selects multiple elements (e.g., h1, p, div).

2) Combinator Selectors

  • Descendant ( ) → Selects all matching elements inside a parent.
  • Child ( > ) → Selects direct children only.
  • Adjacent ( + ) → Selects the immediate next sibling.
  • General Sibling ( ~ ) → Selects all siblings after the element.

section ~ div { background-color: black; color: white; }

h1 + p { font-weight: bold; }

3) Pseudo-Class Selectors ( : )

  • :link → Styles unvisited links.
  • :visited → Styles visited links.
  • :hover → Styles elements when hovered.
  • :active → Styles active links/buttons.
  • :first-child → Selects the first child.
  • :last-child → Selects the last child.
  • :nth-child(n) → Selects the nth child.
  • :first-of-type → Selects the first element of its type.
  • :last-of-type → Selects the last element of its type.
  • :nth-of-type(n) → Selects the nth element of its type.

4) Pseudo-Element Selectors ( :: )

  • ::first-line → Styles the first line of text.
  • ::first-letter → Styles the first letter.
  • ::before → Inserts content before an element.
  • ::after → Inserts content after an element.
  • ::selection → Styles selected text.
  • ::marker → Styles list markers.

5) Attribute Selectors

  • [attribute] → Selects elements with a specific attribute.
  • [attribute="value"] → Selects elements with exact attribute value.
  • [attribute^="value"] → Selects elements whose attribute starts with a value.
  • [attribute$="value"] → Selects elements whose attribute ends with a value.
  • [attribute*="value"] → Selects elements whose attribute contains a value.

input[type="text"] { border: 1px solid black; }
